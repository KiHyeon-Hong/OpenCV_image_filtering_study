Harr Cascades를 이용한 얼굴 검출

  • 눈, 코, 입 검출 데이터 세트 수집을 위해 기존에 개발된 알고리즘을 이용한다.
  • Harr Cascades는 머신러닝 기반의 오브젝트 검출 알고리즘
  • Harr Cascades를 통하여 정면의 이미지에서 눈, 코, 입을 분류하는지 여부를 확인한다.

Harr Cascades를 이용한 얼굴 검출 알고리즘

  • haarcascade_frontalface_alt.xml: 얼굴 검출
  • haarcascade_eye.xml: 눈 검출
  • 정면 이미지에서 얼굴을 먼저 검출한 후에 그 안에서 눈을 검출

face_cascade = cv2.CascadeClassifier('./wei/haarcascade_frontalface_alt.xml')
eye_cascade = cv2.CascadeClassifier('./wei/haarcascade_eye.xml')

img = cv2.imread('./faces/f4001.jpg')
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

faces = face_cascade.detectMultiScale(gray, 1.3, 5)

for (x,y,w,h) in faces:
    cv2.rectangle(img,(x,y),(x+w,y+h),(255,0,0),2)
    roi_gray = gray[y:y+h, x:x+w]
    roi_color = img[y:y+h, x:x+w]
    eyes = eye_cascade.detectMultiScale(roi_gray)
    for i, (ex,ey,ew,eh) in enumerate(eyes):
        if len(eyes) - i < 3:
            cv2.rectangle(roi_color,(ex,ey),(ex+ew,ey+eh),(0,255,0),2)
        
plt.imshow(img)
cv2.destroyAllWindows()
  • 그러나, Harr Cascades 알고리즘의 한계점은 얼굴과 눈은 검출할 수 있지만, 코와 입은 검출할 수 없다.
  • 따라서, Harr Cascades가 아닌 다른 방법을 통해 눈, 코, 입을 검출하도록한다.

결론

  • 인공 신경망을 통한 이미지 변환은 많은 연구가 진행되어 왔다.
  • 이러한 점에서 인공 신경망을 통한 눈, 코, 입 탐지를 시도하려고 하였다.
  • 입력으로 사용할 원본 이미지와, 출력으로 사용할 눈, 코, 입이 표시되어 있는 변환된 이미지를 인공신경망에 학습 시켜 눈, 코, 입을 표시하는 모델을 설계하였다.
  • 눈, 코, 입 표시를 위해 기존에 제공되는 Harr Cascades와 MTCNN을 사용해보았다.
  • MTCNN을 적용한 이미지를 데이터 세트로 저장하였다.
  • 해당 데이터의 크기를 입력받고 출력하는 모델을 설계하였다.
  • 매우 많은 가중치를 계산해야 하므로 개인 서버 환경에서는 학습이 불가능하였다.


  • 순수한 인공 신경망을 통한 눈, 코, 입 식별은 완전 연결 모델 특성 상 학습하는 데에 매우 많은 시간이 걸릴 것으로 예측되어, 적절하지 못하다고 판단된다.
  • 실제 모델을 사용하기 위해서는 전처리를 통하여 용량을 줄일 필요가 있으며, 적절한 필터링을 통해 눈, 코, 입을 추출하기 위한 특징 추출이 매우 중요해 보인다.
